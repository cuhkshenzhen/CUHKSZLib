Detailed Description

Samples the random integer according to Binomial distribution.

It has parameters n as the number of Bernoulli trials and p as the probability of success.

PMF:
\(f(x) = {n \choose x}p^x(1-p)^{n-x}\)

Usage:

BinomialDistribution dist(5, 0.2);
dist.next();
