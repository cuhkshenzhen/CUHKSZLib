◆ genericBinaryPow()

BaseType cuhksz::genericBinaryPow ( BaseType  base,
IntType  exp 
)

Binary power algorithm for all types.

Binary power algorithm, or Exponentiation by squaring is a method to compute the integer power of large integers fast and with high precision.

This function applies to all kinds of types that implement * operation (e.g. Fraction), not only integers. Because all types do not have the same way to construct a "one" object (e.g. identity matrix of Matrix), it cannot calculate the 0 power. If you use integer literal as base argument, you may need to annotate the type of it like 2LL, otherwise, the result may overflow the default integer type int.

Template Parameters
BaseTypeAny type that implements * operation
IntTypeAn ordinary integer type like int, short, etc.
Parameters
baseThe base "number" to be powered
expThe exponent, should be positive
Returns
The powered "number", the same type as base
See also
binaryPow(int64_t base, int exp)

Least common multiple function.

The result is simply calculated by a * b / gcd(a, b). Parameters a and b should be both integers, and the return type is the more precise type in a and b.

Compute the sample variance for a double array.

The sample variance is defined as:

\[\frac{1}{N-1}\sum_{i=0}^{N-1}(x_i-\mu)^2\]

where \(N\) is size, \(\mu\) is mean(arr, size), and \(x_i\) is arr[i].

double cuhksz::variance ( Container  arr)

Compute the population variance for elements in a container object.

The population variance is defined as:

\[\frac{1}{N}\sum_{i=0}^{N-1}(x_i-\mu)^2\]

where \(N\) is arr.size(), \(\mu\) is mean(arr), and \(x_i\) is arr[i].
