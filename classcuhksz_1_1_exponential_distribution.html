Detailed Description

Samples the random real number according to Exponential distribution.

It has parameter lambda, the rate. The Exponential distribution is a special case of Gamma distribution as \(k=1\) and \(\theta=1/\lambda\). So this class is also a subclass of GammaDistribution.

PDF:
\(f(x)=\lambda e^{-\lambda x}\)

where \(\lambda\) is the parameter lambda.

Usage:

ExponentialDistribution dist(1.5);
dist.next();
